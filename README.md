# 💊 Pharmacy Management System

A full-stack Pharmacy Management System built using Node.js, Express, and MySQL with AI-powered insights.

## 🚀 Features

* Role-based login (Admin & Cashier)
* Medicine management (add, update, delete)
* Customer management with phone validation
* Supplier management
* Sales tracking with automatic stock updates

## 🧠 AI Features

* Medicine recommendation based on sales patterns
* Low stock alerts
* Expiry alerts (next 30 days)
* Expired medicines tracking

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express.js
* Database: MySQL
* Authentication: JWT

## ⚙️ Setup

Clone the repository:
git clone https://github.com/nandanrr/pharmacy-management-system.git

Install dependencies:
npm install

Create .env file:
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=pharmacy_db

Run the server:
node server.js
